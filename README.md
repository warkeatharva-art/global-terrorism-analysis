# Global Terrorism Analysis Dashboard (Power BI)

## 📊 Project Overview
This project analyzes global terrorism data using Power BI to uncover patterns, trends, and the overall impact of terrorist activities across different regions and time periods. The dashboard provides an interactive and visual approach to understanding complex data.

---

## 🎯 Objectives
- Analyze global terrorism incidents over time  
- Identify high-risk regions and countries  
- Understand attack types and patterns  
- Evaluate the impact based on casualties  

---

## 📁 Dataset
- Global Terrorism Dataset  
- Contains:
  - Incident date and year  
  - Country and region  
  - Attack type and target  
  - Number of deaths and injuries  

---

## 🛠 Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Excel (Data Cleaning & Transformation)  

---

## 📈 Dashboard Features
- Interactive maps for region-wise analysis  
- Year-wise trend analysis of incidents  
- Country and region filters (slicers)  
- Attack type categorization  
- Casualty analysis (deaths & injuries)  
- Dynamic charts and KPI cards  

---

## 🔍 Key Insights
- Identified regions with the highest number of terrorist incidents  
- Observed increasing/decreasing trends over specific time periods  
- Analyzed most common attack types globally  
- Evaluated the overall impact using casualty data (deaths & injuries)  

---

1. Download the `.pbix` file  
2. Open using Power BI Desktop  
3. Use filters and slicers to explore insights

This project demonstrates how data analytics and visualization can be used to analyze global issues and extract meaningful insights to support informed decision-making.
