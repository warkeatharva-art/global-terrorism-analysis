# Global Terrorism Analysis (Power BI)

## 📊 Project Overview
This project analyzes global terrorism incidents using Power BI.

## 🔍 Key Insights
- Identified attack patterns and regional trends
- Analyzed 27,177 records of incidents
- Visualized data using maps, charts, and slicers

## 🛠 Tools Used
- Power BI
- Excel
- Data Cleaning & Visualization
